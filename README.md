# Stockholic.cpp
data analysis project

## About Program
 
 	- 한정된 주식시장에서 집중투자와 분산투자 포트폴리오의 기대 수익률/리스크 분석 및 비교 


## Feature
 
 	- 0/1 배낭문제 (Knapsack) 알고리즘을 주식 시장에 매수/매도 개념으로 활용 


## Reference
 
	- 현대 포트폴리오 이론 ( https://terms.naver.com/entry.naver?docId=3570360&cid=58783&categoryId=58783 )


## Data Source
